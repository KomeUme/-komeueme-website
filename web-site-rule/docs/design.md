# design.md

## 作風指針（最優先）
- 木版画の空気感を重視する
- 白の余白を積極的に使う
- 情報量を増やしすぎない
- 作品の鑑賞体験を優先する
- SNS的な派手さより、静かな没入感を優先する

## 基本デザイントーン
- 背景は `--paper: #f4f4f2` を基調とし、淡いノイズ状の radial-gradient を重ねる
- 境界線は `--line: #d9d9de` を基本に、白背景カード＋薄い枠線で面を分ける
- 角丸はカード系で `8px`、ボタンは `999px`（ピル形）を使う

## タイポグラフィ運用
- 本文系フォント: `"Helvetica Neue", Arial, "Hiragino Sans", "Yu Gothic", sans-serif`
- 見出し系フォント: `"Didot", "Bodoni 72", "Times New Roman", "Hiragino Mincho ProN", "Yu Mincho", serif`
- 明朝本文（作品説明）: `"Hiragino Mincho ProN", "Yu Mincho", serif`
- 基本行間は `body: 1.7`、キャプション本文は `1.55` を基準とする

## 代表サイズ（現行実装ベース）
- ブランドロゴ: `54px`（<=760px: `42px`, <=520px: `34px`）
- ナビ文字: `13px`、サブメニュー: `12px`
- ヒーロー見出し: `clamp(41px, 5.8vw, 67px)`（<=520pxは `clamp(24px, 6.8vw, 32px)`）
- セクション見出し: `20px`
- 本文小テキスト: `13-14px`

## レイアウト共通
- `main` は `max-width: 1180px; margin: 0 auto; padding: 20px 20px 86px`
- PCでは Grid/Flex の複数カラムを使い、モバイル（`760px`以下）で基本1カラム化する。ただし作品一覧は例外で、比較性と導線性を優先して横3作品を基本にする
- 全ページに固定言語スイッチ（右上）を置く

## レスポンシブ閾値
- ブレークポイントは `760px` と `520px` の2段階
- `760px`以下: 通常グリッドは1カラム化、ヒーロー画像高さと余白を圧縮。作品一覧のみ横3作品を基本にする
- `520px`以下: タイトルとヒーロー領域をさらに縮小する。作品一覧の横3作品基本は維持し、極端な縦長/横長作品だけ2列分を使ってよい
